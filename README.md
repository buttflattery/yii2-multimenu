# yii2-completemenu
a yii2 menu extension used for creating left menu or top nav


##running migrations
- windows
`php yii migrate/up --migrationPath=@vendor/buttflattery/yii2-multimenu/src/migrations`
- ubuntu 
`php ./yii migrate/up --migrationPath=@vendor/buttflattery/yii2-multimenu/src/migrations`
